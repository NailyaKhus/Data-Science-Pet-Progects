# Binance_test
Pulling data out of Binance and predicting the cryptocurrency quotes

В файле **parsing through GET.ipynb** представлен парсер котировок с сайта Binance.com. 
Скачана котировка USDTRUB.
Временной промежуток с 00:00:00 часов 01.05.2022 по 06:00:00 часов 03.05.2023 с промежутком в 1 час. 

В файле **Prediction.ipynb** представлено предсказание параметра Close price (цена закрытия) с помощью LSTM.
Выполнено:
- очистка датасета (удаление дублей, восстановление пропусков с помощью интерполяции);
- разбиение данных на трейн, валидацию и тест;
- нормализация данных;
- создание объекта tf.data.Dataset с помощью timeseries_dataset_from_array для удобного разбиения на батчи;
- построена нейронная сеть;
- применены методы объекта tf.keras.callbacks (EarlyStopping - для остановки обучения в случае отсутствия улучшений в метриках, ReduceLROnPlateau - для подбора параметра learning_rate);
- подобраны гиперпараметры (hidden_input, batch_size, bias_regularizer, kernel_regularizer, activity_regularizer);
- визуализация полученных результатов.

Пути возможного улучшения:
1.  - декомпозировать временной ряд на тренд, сезон, цикличность и шум;
    - предсказать тренд, сезон, цикличность по отдельности;
    - объединить предсказания (либо с помощью блендинга или ансамбля с мета-моделью, либо сделать нейронку multi input - one output);
2.  - проверить ряд на стационарность;
    - применить статистические методы предсказания временных рядов (например, модель Хольта-Винтерса);
3.  - добавление признаков - feature engineering (временные сигналы, признаки роста/падения цены на предыдущем периоде, среднюю за предыдущий период и тд.) и построение модели уже на временном ряду с признаками.
